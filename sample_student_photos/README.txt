INSTRUCTIONS FOR STUDENT PHOTOS
==================================================

1. Place student photos in this folder
2. Name each photo with the student's name:
   - Basistha.jpg
   - Student1.png
   - Student2.jpg
   etc.

3. After adding all photos, compress this folder
4. Rename the ZIP file to 'name.zip'
5. Place 'name.zip' in the main project folder

Note: Photos should be clear, well-lit headshots
